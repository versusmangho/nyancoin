# nyancoin
이 웹앱은 오버필드 게임의 냥코인 효율을 계산하고, 게임 데이터를 직접 관리할 수 있는 도구입니다.

**주요 기능:**

1.  **냥코인 효율 계산기:**
    *   아이템의 제작/납품 효율을 자동으로 계산해 줍니다.
    *   `마지노선 효율`, `절약 레벨`, `워라밸 레벨`, `스태미나 1당 가치` 등 다양한 게임 설정을 조절하여 최적의 효율을 찾아낼 수 있습니다.
    *   **동적 스태미나 가치:** 워라밸 레벨에 따라 스태미나 1당 냥코인 가치가 자동으로 계산되거나, 직접 설정할 수도 있습니다.
    *   **중간재료 스태미나 미포함:** 중간 재료의 스태미나 소모를 계산에 포함할지 여부를 선택할 수 있습니다. (기본 활성화)
    *   **계단식 납품 시스템:** 납품 횟수에 따라 요구되는 아이템 개수가 달라지는 게임 시스템을 반영하여 정확한 효율을 제공합니다.
    *   모든 설정 및 입력값 변경 시 실시간으로 계산 결과와 총 요약이 업데이트됩니다.

2.  **재료 및 레시피 관리 (데이터 에디터):**
    *   `data.json` 파일에 저장된 게임 내 모든 재료 및 레시피 정보를 웹에서 직접 추가, 편집, 삭제할 수 있습니다.
    *   재료의 상점 가격, 레시피의 필요 스태미나, 재료 목록, 그리고 카테고리(가공품, 방직 등) 등을 손쉽게 관리할 수 있습니다.
    *   편집된 데이터는 `data.json` 파일로 다운로드하여 로컬에 저장하고 다시 불러와 사용할 수 있습니다.

**사용법:**
*   `index.html`을 열어 계산기를 사용합니다.
*   `editor.html`을 열어 재료 및 레시피 데이터를 관리합니다.
*   `data.json` 파일을 동일한 폴더에 두면 웹앱이 자동으로 데이터를 로드합니다. (로컬 테스트 모드에서 파일을 직접 선택할 수도 있습니다.)

이 리포지토리의 거의 모든 내용은 Google Gemini를 통해 생성되었습니다.
